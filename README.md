# Homebrew Tap: rss-reader

This tap provides a Homebrew formula for the TUI RSS reader.

## Install
```bash
brew tap MOHY-Li/homebrew-tap
brew install rss-reader
```

## Run
```bash
rss-reader
```

## Notes
- The formula builds from source with Rust.
- If you need to update the formula, edit `Formula/rss-reader.rb`.
